# PT-BR | Imersão Dados com Python II (Fev, 2026)
## Sobre o Projeto
Este repositório contém o resultado do trabalho de 4 dias intensivos de Análises de Dados promovido pela **Alura**. O foco foi desenvolver o pipeline completo de dados: desde a manipulação de raw data (dados brutos) até a entrega de valor através de um dashboard interativo hospedado via Streamlit.
## Habilidades Técnicas Desenvolvidas
**Python para Análise de Dados:** Sintaxe aplicada à resolução analítica de problemas.    
**Manipulação de Dados com Pandas:** Limpeza, transformação, e preparo complexo de Dataset (Data Wrangling).    
**Computação Numérica com Numpy:** Uso de álgebra linear e operações matriciais para estatísticas descritivas e manipulação eficiente de grandes volumes de dados.   
**Vizualição Profissional de Dados:** Criação de gráficos dinâmicos e interativos utilizando a biblioteca Plotly.     
**Data Apps com Streamlit** Desenvolvimento de aplicações web para transformar scripts de análise em ferramentas acessíveis ao usuário final. 
## Recursos do Dashboard
O dashboard foi projetado para oferecer três niveis de profundidade:
* Visão Geral
  - Métricas principais em tempo real
  - KPI's (Indicadores-chave de desempenho)
  - Tendências temporais
* Análise Detalhada
  - Filtros por período, categoria e região
  - Comparativos ano a ano
  - Detalhamento por segmentos  
* Visualizações
  - Gráfico de barras com drill-down  
  - Linhas temporais interativas
  - Mapas de calor e correlação  
  - Distribuições estatísticas
## Como executar o Projeto
1. Clone o repositório:
```bash
git clone https://github.com/leandroollem/python-imersao-dados
```
2. Instale as dependências:
``` bash
pip install -r requirements.txt
```
3. Execute o dashboard:
``` bash
streamlit run app.py
```

O certificado de participação foi emitido pela Alura, validando a conclusão de todos os módulos técnicos.

# EN-US | Data Immersion with Python II (Feb, 2026)
## About the Project
This repository contains the results of a 4-day intensive Data Analysis program promoted by Alura. The focus was on developing a full data pipeline: from raw data manipulation to delivering value through an interactive dashboard hosted via Streamlit.
## Technical Skills Developed
**Python for Data Analysis:** Syntax applied to analytical problem-solving and process automation.  
**Data Manipulation with Pandas:** Cleaning, transforming, and complex dataset preparation (Data Wrangling).  
**Numerical Computing with NumPy:** Use of linear algebra and matrix operations for descriptive statistics and efficient manipulation of large datasets.  
**Professional Data Visualization:** Creation of dynamic and interactive charts using the Plotly library.  
**Data Apps with Streamlit:** Web application development to transform analysis scripts into tools accessible to the end-user.  
## Dashboard Features
The dashboard was designed to offer three levels of depth:
* Overview
  - Key real-time metrics (summary cards).
  - KPIs (Key Performance Indicators) tracking.
  - Temporal trends and patterns.
* Detailed Analysis
  - Dynamic filters by period, category, and region.
  - Year-over-Year (YoY) comparisons.
  - Business segment breakdown.

* Advanced Visualizations
  - Bar charts with drill-down functionality.
  - Interactive timelines.
  - Heatmaps and correlation analysis.
  - Statistical distributions (Histograms and Boxplots).
